🔥 إعداد Firebase — التخزين السحابي

⚠️ Firebase غير مُعدّ — البيانات محفوظة محلياً فقط (لا تتزامن بين الأجهزة)
📋 خطوات إعداد Firebase (مرة واحدة فقط):
  1. اذهب إلى console.firebase.google.com
  2. أنشئ مشروعاً جديداً ← اختر اسماً ← أكمل الإنشاء
  3. من الصفحة الرئيسية ← أضف تطبيق ويب (</>)
  4. انسخ كائن firebaseConfig والصقه أدناه
  5. من القائمة الجانبية: Build ← Firestore Database ← Create
  6. اختر Start in test mode ← أكمل
📋 الصق كود firebaseConfig هنا (الطريقة السريعة):
يمكنك لصق الكود كاملاً — سيتم استخراج القيم تلقائياً
✏️ أو أدخل القيم يدوياً:
🔒 الأمان: يُخزَّن الإعداد في متصفحك المحلي فقط. لا يُرسَل لأي جهة خارجية. قواعد Firestore الافتراضية (test mode) مناسبة للاستخدام الداخلي.
